# 1️⃣ 개념 정리 
## 01. 데이터를 분석하다

```
✅ 학습 목표 :
* 데이터에 대해 이해한다.
* 데이터 분석에 대해 이해한다.
```

```
1. 데이터 : 수치의 모음 

2. 데이터 분석의 목적
1) 데이터 요약
2) 데이터 설명 : 대상이 가진 성질과 관계성을 명확히 밝히고 이를 이해함
3) 새로 얻을 데이터 예측 : 이미 얻은 데이터를 기반으로, 이후 새롭게 얻을 데이터 예측

2. 인과관계 vs 상관관계
인과관계 : 2가지 중 하나(원인)을 변화시키면, 다른 하나(결과)도 바꿀 수 있는 관계
상관관계 : 한쪽이 크면 다른 한쪽도 큰(또는 한쪽이 크면 다른 한쪽은 작은) 관계
```

## 02. 통계학의 역할

```
✅ 학습 목표 :
* 산포에 대해 설명할 수 있다.
```

1. 통계학의 중요 역할 : 퍼짐(산포)이 있는 데이터에 대해 설명이나 예측 수행 
*퍼짐 : 데이터에 포함된 값 하나하나의 차이 

2. 확률론 : 데이터 퍼짐이나 불확실성을 확률로 나타냄 


## 03. 통계학의 전체 모습
```
✅ 학습 목표 :
* 기술통계와 추론통계의 차이점을 설명할 수 있다.
* 통계적 추론과 가설 검정에 대해 이해한다.
```
1. 기술통계 vs 추론통계
기술통계 : 수집한 데이터를 정리하고 요약하는 방법
-> 확보한 데이터에만 집중. 데이터 자체의 성질 이해를 목표로 함
추론통계 : 수집한 데이터로부터 데이터의 발생원을 추정하는 방법

2. 통계적 추론과 가설검정
추론통계의 종류 : 통계적추론, 가설검정
1) 통계적 추론 : 데이터에서 가정한 확률 모형의 성질 추정하는 방법
2) 가설검정 : 세운 가설과 데이터가 얼마나 들어맞는지를 평가하여, 가설 채택할지 판단하는 방법


## 05. 모집단

```
✅ 학습 목표 :
* 모집단의 개념을 명확히 설명할 수 있다.
```

1. 모집단 : 알고자 하는 대상 전체
2. 모집단 크기 : 모집단에 포함된 요소의 수 
1) 유한모집단 : 모집단 중 한정된 요소만 포함한 것
2) 무한모집단 : 모집단 중 포함된 요소의 개수가 무한한 것 

## 06. 모집단의 성질을 알다 

```
✅ 학습 목표 :
* 전수조사와 표본조사의 개념적 차이와 각각의 필요성을 설명할 수 있다.
* 표본에 대해 이해한다.
```

1. 전수조사
: 모집단에 포함된 모든 요소를 조사하는 방법
-> 유한모집단일 때 선택할 수 있는 조사 방법
-> 분석할 데이터 = 모집단
-> 획득한 데이터의 특징을 파악하고 기술만 해도 모집단의 성질을 설명하고 이해 가능[기술통계]

2. 표본조사
표본 : 추론통계에서 조사하는 모집단의 일부
표본추출 : 모집단에서 표본을 뽑는 것
표본조사 : 표본을 이용해 모집단의 성질을 조사하는 것
표본크기 : 표본에 포함된 요소의 개수

# 2️⃣ 확인 문제

## 문제 1.

> **🧚Q. 한 회사가 직원들의 ‘업무 만족도’를 파악하기 위해 설문조사를 실시했다.
전체 직원 1,000명 중 50명에게만 설문을 진행했고, 그 결과 평균 만족도는 4.5점(5점 만점)이었다.
회사는 이를 근거로 “우리 회사 직원들은 대체로 업무 만족도가 높다”고 결론 내리려 한다.**
>
> **이 결론이 타당한지 판단하기 위해 가장 먼저 확인해야 할 것은 무엇인가?**

~~~
1️⃣ 조사 대상 50명이 전체 직원을 대표하는 표본인가?
2️⃣ 만족도의 평균값만으로 결론을 내려도 되는가?
3️⃣ 설문 문항이 객관적으로 구성되었는가?
4️⃣ 이 결과를 모집단(전체 직원)으로 일반화해도 되는가?
~~~



<!--학습한 개념을 활용하여 자유롭게 설명해 보세요.-->

```
전체 직원 1,000명 중 표본으로 추출한 50명은 표본크기로 부족하다고 생각한다. 전체 중 5%에 불과하기 때문에 대표성을 가지지 못한다. 또한, 극단치가 평균을 왜곡한 가능성도 있기 때문에 만족도의 평균값만으로 결론을 내려서는 안된다. 따라서, 이 결과를 모집단으로 일반화하는 건 타당하지 않다.
```
