# 1️⃣ 개념 정리 
## 01. 데이터 유형

```
✅ 학습 목표 :
* 기초적인 통계 용어에 대해 설명할 수 있다.
* 데이터 유형에 대해 이해한다.
```

```
1. 변수
1) 변수 : 공통의 측정 방법으로 얻은 같은 성질의 값

2) 차원 : 변수의 개수 

2. 데이터 유형
1) 양적 변수(수치형 변수) : 숫자로 나타낼 수 있는 변수
(1) 이산형 양적 변수(이산변수) : 얻을 수 있는 값이 점 예) 횟수, 사람 수 등
(2) 연속형 양적 변수(연속변수) : 간격 없이 이어지는 값 예) 키, 몸무게 등 

2) 질적 변수(범주형 변수) : 숫자가 아닌 범주로 값 나타냄 예) 예/아니요, 동전의 앞/뒤 등
-> 변수 간 대소 관계 없음
```

```
✅ 학습 목표 :
* 변수 유형에 따라 히스토그램 정의를 설명할 수 있다.
```

```
1. 데이터 분포 표현 : 도수분포도(히스토그램) 사용

2) 연속형 양적 변수의 히스토그램 : 범위 설정하고, 범위에 포함되는 숫자 개수를 두어 세로축에 둠
-> 구간폭 : 범위의 넓이
3) 범주형 변수의 히스토그램 : 가로축은 각 범주, 세로축은 각 범주에 속하는 개수
```

## 03. 통계량
```
✅ 학습 목표 :
* 통계량의 개념과 역할을 이해한다.
* 대푯값의 종류와 각 대푯값의 특성을 설명할 수 있다.
* 자료의 산포를 나타내는 분산과 표준편차의 의미를 이해한다.
* 이상값(이상치)의 개념과 통계 분석에서의 영향을 이해한다.
```

```
1. 통계량
: 수집한 데이터로 계산을 수행해 얻은 값
1) 기술통계량 또는 요약통계량 : 데이터 그 자체의 성질을 기술하고 요약하는 통계량
-> 주로 양적 변수가 대상

2. 기술통계량의 종류
1) 대푯값 : 대략적인 분포 위치, 즉 대표적인 값을 정량화하기 위해 사용하는 통계량
(1) 평균값 
(2) 중앙값 : 크기 순으로 값을 정렬했을 때 한가운데 위치한 값
(3) 최빈값 : 데이터 중 가장 자주 나타나는 값 

2) 데이터 퍼짐 정도 표현하는 값 : 분산, 표준편차
(1) 표본분산 : 표본의 각 값과 표본평균이 어느 정도 떨어져 있는지를 평가하는 값
-> 상자 수염 그림 : 분산 확인을 위해 주로 사용

-> 분포 형태 다양하게 시각화하는 방법 : 바이올린 플롯, 스웜 플롯 등

3. 이상값
: 평균값에서 표준편차의 2배 또는 3배 이상 벗어난 숫자
```

## 04. 확률

```
✅ 학습 목표 :
* 확률의 기본 개념과 확률변수 및 확률분포의 의미를 이해한다.
* 추론통계에서 사용되는 확률분포의 개념을 이해한다.
* 사건 간 독립의 개념과 통계적 의미를 이해한다.
```

```
1. 확률의 기본 사고방식
1) 확률 : 발생 여부가 불확실한 사건의 발생 가능성을 숫자로 표현한 것
2) 확률변수 : 확률이 달라지는 변수
3) 실현값 : 확률변수가 실제로 취하는 값
4) 확률분포 : 가로축에 확률변수, 세로축에 그 확률변수의 발생가능성을 표시한 분포
(1) 확률밀도함수 : 확률변수가 연속형일 때 확률분포

2. 추론통계와 확률분포
1) 기댓값 : 변수가 확률적으로 얼마나 발생하기 쉬운가를 평균적인 값으로 나타낸 것
2) 분산 : 확률분포가 기댓값 주변에 어느 정도 퍼졌는지를 나타내는 값
3) 표준편차 : 분산의 제곱근을 취한 값
-> 분산과 표준편차의 성질 : 0 이상, 모두 같은 값이 나타나는 경우에는 0, 기댓값에서 떨어진 값이 많을수록 커짐
4) 왜도 : 분포가 좌우대칭에서 어느 정도 벗어났는지
5) 첨도 : 분포가 얼마나 뾰족한지와 그래프의 꼬리가 차지하는 비율 
6) 조건부확률 : 한쪽 확률변수 Y의 정보가 주어졌을 때, 다른 한쪽 확률변수 X의 확률 
```


## 06. 추론통계를 배우기 전에

```
✅ 학습 목표 :
* '데이터를 얻는다는 것'의 의미를 이해한다.
* 무작위추출의 개념과 통계적 필요성을 설명할 수 있다.
* 추론통계의 직관적 의미를 이해한다
```

```
1. 전수조사와 표본조사
1) 전수조사 : 모집단의 모든 요소 조사
2) 표본조사 : 모집단의 일부인 표본으로 모집단의 성질 추정

2. 데이터 추출
1) 모집단에서 일부를 표본으로 추출 = 모집단에 포합된 전체 값으로 구성된 분포에서 일부를 추출
-> 모집단분포 : 모집단을 나타내는 분포
2) 확률분포와 실현값의 관계 : 모집단과 표본의 관계와 매우 비슷함
-> 모집단 = 확률분포, 표본 = 확률분포를 따르는 실현값
3) 모집단분포 모형화 : 수학적인 확률분포로 모집단분포를 근사하는 것
4) 무작위추출 : 데이터 얻을 때 모집단에 포함된 요소를 하나씩 무작위로 선택하여 추출하는 방식
(1) 단순무작위추출법  표본에 있을 수 있는 모든 요소를 목록으로 만들고, 난수를 이용하여 표본을 정하는 것
(2) 층화추출법 : 모집단을 몇 개의 층(집단)으로 미리 나눈 뒤, 각 층에서 필요한 수의 조사대상을 부작위로 추출하는 방법 
```

```
* 신뢰구간의 의미와 활용 방법을 이해한다.
```

```
1. 표본오차 : 정말로 알고 싶은 것과 실제로 손 안에 있는 데이터에 생기는 어긋남(오차)

2. 큰 수의 법칙 : 표본크기 n이 커질수록 표본평균이 모집단평균에 한없이 가까워진다는 법칙

3. 표본오차의 확률분포
1) 중심극한정리 : 모집단이 어떤 분포이든 간에, 표본크기 n이 커질수록 표본평균의 분포는 정규분포로 근사할 수 있다는 것
2) 중심극한정리 : 모집단이 어떤 분포이든 간에, 표본크기 n이 커질수록 표본평균의 분포는 정규분포로 근사할 수 있다는 것
3) 추정량 : 모집단의 성질을 추정하는 데 사용하는 통계량
4) 일치추정량 : 표본크기 n을 무한대로 했을 때, 모집단의 성질과 일치하는 추정량
5) 비편향추정량 : 추정량의 평균값(기댓값)이 모집단의 성질과 일치할 때의 추정량
-> 매번 얻을 때마다 확률적으로 다른 값이 되지만, 평균으로 보면 모집단의 성질을 과대하지도 과소하지도 않게 나타내는 양을 뜻함

4. 신뢰구간
1) OO% 신뢰구간 : OO%의 확률로 이 구간에 모집단평균이 있다
2) t분포 : 모집단이 정규분포라는 가정하에 미지의 모집단 표준편차를 표본으로 계산한 비편향표준편차로 대용했을 때, (x - u)를 표준오차로 나누어 표준화한 값이 따르는 분포
3) 정밀도를 높이려면? 표준오차에 주목. 분자인 비편향표준편차를 작게 하거나, 분모인 표본크기를 크게 함
```

# 2️⃣ 확인 문제

## 문제 1.

> **🧚Q. 한 기업이 신제품 출시 후 “소비자 전반의 만족도”를 파악하고자 한다.
이를 위해 전체 소비자 중 일부를 무작위로 선택해 설문을 실시했고, 평균 만족도가 높게 나타났다.
그러나 내부 회의에서 다음과 같은 의견이 나왔다.

“설문 결과가 좋으니, 우리 제품은 소비자 전체에게도 만족도가 높다고 결론 내려도 되지 않을까?”
> **이 결론의 타당성을 판단하기 위해 통계학적으로 가장 적절한 설명은 무엇인가?**

~~~
1️⃣ 일부 소비자를 조사했으므로 이는 전수조사이며, 결과는 그대로 신뢰할 수 있다.
2️⃣ 표본의 평균이 높게 나타났다면, 모집단의 평균도 반드시 높다고 볼 수 있다.
3️⃣ 표본이 모집단을 적절히 대표한다면, 표본 결과를 통해 모집단의 특성을 추론할 수 있다.
4️⃣ 데이터 분석의 목적은 결과를 빠르게 도출하는 것이므로 표본의 대표성은 중요하지 않다.
~~~

```
2번 : 표본의 평균이 높게 나타났다고 해서, 모집단의 평균이 반드시 높다고는 볼 수 없다. 표본은 무작위로 추출되므로 추출된 표본에 따라 값이 변하기 때문이다.
4번 : 표본이 대표성이 있어야 데이터 분석 역시 의미 있다.
```
