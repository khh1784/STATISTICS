# 1️⃣ 개념 정리 
## 01. 가설검정의 원리

```
✅ 학습 목표 :
* 가설검정의 원리에 대해 설명할 수 있다.
* 통계적에서의 가설을 이해한다.
```
```
1. 가설 검증법
: p값(p-value) 수치 계산
1) 확증적 자료분석(가설검증형 데이터 분석) : 미리  세운 가설을 검증하는 접근법
2) 탐색적 자료분석 : 가설을 미리 세우지 않고, 전체 데이터를 탐색적으로 해석하는 접근법

2. 가설검정
: 데이터에 기반을 둔 통계 가설을 검증하는 방법
1) 실험군 : 어떠한 조치를 취한 집단
2) 대조군 : 실험군과 비교 및 대조를 위해 마련한 집단

3. 귀무가설과 대립가설
1) 귀무가설 : 밝히고자 하는 가설의 부정 명제
2) 대립가설 : 밝히고 싶은 가설
-> 흐름 : 부정 명제인 귀무가설을 세우고, 이 귀무가설이 틀렸음을 주장하는 것으로 대립가설 지지

4. p값
: 귀무가설이 옳다고 가정했을 때 관찰한 값(예 : 평균값의 차이) 이상으로 극단적인 값이 나올 확률
-> 귀무가설과 현실 데이터 간의 괸리 정도 평가
1) p값이 0.05 이하인 경우 : 귀무가설 하에서 현실 데이터가 나타나기 어려움 -> 귀무가설 기각 및 대립가설 채택
=> 통계적으로 유의미한 차이 존재
2) p값이 0.05 상회하는경우 : 귀무가설을 기각할 수 없음
=> 통계적으로 유의미한 차이 발견하지 못함 
```

## 03. 가설검정 관련 그래프
```
✅ 학습 목표 :
* 오차막대에 대해 설명할 수 있다.
* '통계적으로 유의미'의 의미를 이해한다.
```
```
1. 오차 막대
: 반복이 있는 데이터에서 평균값을 계산하여 막대그래프나산점도 그릴 때 평균값에 더하여 그 위아래로 오차 막대 그림
-> 오차 막대가 무엇을 표시하는지 그래프 범례에 반드시 기재하기

2. 오차 막대 목적
1) 평균값+- 표준오차(SEM) : 평균값의 확률을 나타내고 싶을 때
2) 평균값 중심의 95% 신뢰구간 : 신뢰구간 나타내고 싶을 때
3) 평균값 +- 표준편차 : 데이터의 퍼짐 나타내고 싶을 때  

3. 통계적으로 유의미를 나타내는 표기
: *(별표)를 일반적으로 사용
```

## 04. 제1종 오류와 제2종 오류

```
✅ 학습 목표 :
* 제1종 오류와 제2종 오류의 개념을 구분하고 설명할 수 있다.
* 유의수준 α와 제1종 오류의 관계를 이해한다.
* 제2종 오류 β와 검정력(1-β)의 의미를 이해한다.
* α와 β의 상충 관계를 설명할 수 있다.
* 표본크기와 효과크기가 검정력에 미치는 영향을 이해한다.
```

```
1. 제1종 오류
: 실제로는 아무런 차이가 없음에도 차이가 있다고 판단해 버리는 잘못
-> 해결방안 : 유의수준의 값을 미리 정해 둠으로써, 제 1종 오류가 일어날 확률 통제 가능

2. 제2종 오류
: 정말 차이가 있는데도 차이가 있다고는 말할 수 없어, 귀무가설을 기각하지 않는 판단을 내려 버리는 것
cf) 검정력 : 제 2종 오류가 일어나지 않는 확률. 즉, 정말 차이가 있을 때 차이가 있다고 올바르게 판단할 확률
-> 제 2종 오류가 일어나는 확률은 직접 통제 불가능. 이는 표본크기가 커질수록 작아지며, 어느 정도의 차이를 차이로 간주하는지를 나타내는 값인 효과크기가 커짐에 따라서도 작아짐

3. 제 1종 오류가 일어나는 확률과 제 2종 오류가 일어나는 확률의 관계
: 상충 관계(한쪽이 작아지면 또 다른 한쪽은 커짐)
-> 표본크기가 클수록, 정말로 차이가 있을 때에 그렇다고 판단할 확률인 검정력이 상승

4. 효과크기
: 일반적으로 얼마나 큰 효과가 있는지를 나타내는 지표
-> 평균값의 차이에 비해 표준편차가 클수록 2개 분포의 겹치는 부분이 커지므로, 효과크기는 작아지고 평균값의 차이는 검출하기 어려워짐
-> 평균값의 차이에 비해 표준편차가 작을수록 2개 분포의 겹치는 부분은 작아지므로, 효과크기는 커짐

5. 가설검정 시
: 모집단 대상으로 검출하고 싶은 효과크기 미리 설정하는 것이 바람직함
-> 제 1종 오류가 일어나는 확률, 제 2종 오류가 일어나는 확률, 표본크기, 효과크기의 네 값 중 셋 결정 시 나머지 하나는 자동으로 정해짐 
```


## 05. 다양한 가설검정

```
✅ 학습 목표 :
* 가설검정 방법의 구분 요령을 이해한다.
* 다양한 가설검정 방법에 대해 설명할 수 있다.
```

```
1. 가설검정 해석의 흐름
1) 확인하고 싶은 대상에 따라 귀무가설과 대립가설 설정
2) 데이터로 가설검정에 필요한 검정통계량 계산
3) 귀무가설이 옳다는 가정하에 통계량의 분포 생각하고, 데이터로 계산한 통계량이 분포의 어느 위치에 있는지를 구하여 p값 계산
-> 가설검정 방법 선택 시 데이터 유형, 표본 수, 양적 변수 분포 성질 확인하기

2. 분할표
: 각 상태에 몇 개 데이터가 있는지를 나타내는 표
-> 데이터 유형에 따라 해석 방법이 다르니 데이터 유형부터 확인하기

3. 양적 변수의 성질
1) 모수검정 : 모집단이 수학적으로 다룰 수 있는 특정 분포를 따른다는 가정을 둔 가설검정
-> 대부분 모집단의 분포가 정규분포임
+) 정규성 : 데이터가 정규분포로부터 얻어졌다고 간주할 수 있는 성질
2) 비모수검정 : 평균이나 표준편차 등의 파라미터(모수)에 기반을 두지 않는 바업

```


## 06. 대푯값 비교

```
✅ 학습 목표 :
* 모수 검정의 평균값 비교에 대해 설명할 수 있다.
* 적절하지 않은 검정을 사용하면 어떻게 되는지 설명할 수 있다.
* 비모수 검정의 대푯값 비교에 대해 설명할 수 있다.
* 자유도와 분산분에 대해 설명할 수 있다.
* 다중비교 검정에 대해 설명할 수 있다.
```

```
1. 일표본 t검정
: 어떤 평균값의 모집단에서 표본을 얻었는가를 조사

2. 이표본 t검정
: 2개 집단의 평균값 비교
-> t검정 : 모수검정으로 분류되는 검정방법이므로 데이터에 정규성 필요
일반적으로 t검정에서는 등분산성 가정
cf) 분산 미일치 : 웰치의 t검정 이용

3. 정규성 조사
1) Q-Q 플롯(분위수-분위수 그림)
2) 샤피로-윌크 검정 : 가설검정으로 조사
3) 콜모고로프-스미르노프(K-S) 검정 : 이론적인 분포와 비교

4. 등분산성 조사
: 바틀렛 검정, 레빈 검정

5. 비모수검정의 대푯값 비교
1) 비모수 버전의 2개 표본 대푯값 비교
: 각 집단 데이터에 정규성 없는 경우에는 비모수검정으로 분류되는 방법 사용이 권장
-> 대표적 방법 : 윌콕슨 순위합 검정(평균값 대신 각 데이터 값의 순위에 기반해 검정 실시), 맨-휘트니 U 검정, 플리그너-폴리셀로 검정, 브루너0문첼 검정
2) 분산분석
: 3개 이상 집단의 평균값을 비교하는 방법
-> F값 계산해 검정통계량 만듦. 귀무가설이 올바르다는 가정하에, F분포 따름.
p값 : 이 분포에서 관찰한 F값 이상으로 극단적인 값이 나올 확률. 
 
```
